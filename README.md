# 🌦️ Weather App

A simple Python-based Weather Application that fetches real-time weather data using an API and stores it in a database.

---

## 🚀 Features
- Enter city name
- Fetch real-time weather data
- Display temperature and humidity
- Store results in SQLite database

---

## 🛠️ Technologies Used
- Python
- API (OpenWeatherMap)
- SQLite
- JSON

---

## 📂 Project Structure
weatherapp/
│── main.py
│── database.py
│── cities.txt
│── requirements.txt
│── README.md

---

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
py main.py

---

Enter city name: Chennai  
Temperature: 30°C
